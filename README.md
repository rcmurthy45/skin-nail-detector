## 🛠️ STEP 1 — Install Python & Dependencies

Make sure you have **Python 3.9–3.11** installed.

```bash
# Create a virtual environment (recommended)
python -m venv venv

# Activate it
# Windows:
venv\Scripts\activate
# Mac/Linux:
source venv/bin/activate

# Install all dependencies
pip install -r requirements.txt
```

---

## 📦 STEP 2 — Download Kaggle Datasets

You need images for 4 disease classes. Here are recommended free Kaggle datasets:

### Option A — Skin Diseases (Acne, Eczema, Psoriasis)
🔗 https://www.kaggle.com/datasets/shubhamgoel27/dermnet

### Option B — Nail Fungus
🔗 https://www.kaggle.com/datasets/chatsith/nail-fungus-dataset
(Or search "nail fungus" on Kaggle)

### How to Download from Kaggle:

**Method 1 — Manual (Easiest):**
1. Go to https://www.kaggle.com and create a free account
2. Visit each dataset URL above
3. Click **"Download"** button (downloads a .zip file)
4. Extract the zip file

**Method 2 — Kaggle CLI:**
```bash
pip install kaggle

# Place your kaggle.json API key in ~/.kaggle/kaggle.json
# (Download from Kaggle > Account > API > Create New Token)

kaggle datasets download -d shubhamgoel27/dermnet
unzip dermnet.zip -d temp_dermnet/
```

### After Downloading — Organize into this structure:
```
dataset/
  acne/          ← copy ~200-500 acne images here
  eczema/        ← copy ~200-500 eczema images here
  psoriasis/     ← copy ~200-500 psoriasis images here
  nail_fungus/   ← copy ~200-500 nail fungus images here
```

> 💡 **Tip:** More images = better accuracy. Aim for at least 100 images per class.
> For best results: 200-500+ images per class.

---

## 🧠 STEP 3 — Train the AI Model

```bash
python train_model.py
```

**What this does:**
- Loads images from `dataset/` with data augmentation
- Trains MobileNetV2 (transfer learning) for 20 epochs
- Saves the best model to `model/skin_nail_model.h5`
- Generates an accuracy/loss graph at `model/training_history.png`

**Expected training time:**
- CPU only: ~30–60 minutes
- With GPU: ~5–15 minutes

## 📊 Model Architecture

```
Input Image (224×224×3)
       ↓
MobileNetV2 (pretrained on ImageNet, frozen initially)
       ↓
GlobalAveragePooling2D
       ↓
Dense(256) → BatchNorm → Dropout(0.4)
       ↓
Dense(128) → Dropout(0.3)
       ↓
Dense(4) → Softmax
       ↓
Output: [acne, eczema, nail_fungus, psoriasis]
```

## 🐛 Troubleshooting

**"Model not found" warning:**
→ Run `python train_model.py` first, or the app runs in demo mode.

**TensorFlow import error:**
→ Run `pip install tensorflow`

**"No module named flask":**
→ Activate your virtual environment: `venv\Scripts\activate` (Windows)

**Voice input not working:**
→ Use Google Chrome or Microsoft Edge. Firefox has limited Web Speech API support.

**Low accuracy:**
→ Add more images per class. Use at least 200+ images per disease category.
